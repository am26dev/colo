# Imagens da Colo

Coloca aqui as fotos do site. Sugestões de nomes (são os que o site procura):

| Ficheiro            | Onde aparece                          | Tamanho sugerido |
|---------------------|---------------------------------------|------------------|
| `logo.svg` / `.png` | logótipo oficial (opcional)           | vetorial / 600px |
| `hero.jpg`          | imagem grande no topo                 | 1000 × 1250 px   |
| `og-cover.jpg`      | pré-visualização ao partilhar o link  | 1200 × 630 px    |
| `salmao.jpg` …      | fotos dos pratos (1 por prato)        | 800 × 550 px     |

## Como ligar uma foto a um prato
No ficheiro `data/menu.js`, no prato, preenche o campo `foto`:

```js
foto: "assets/img/salmao.jpg"
```

> Enquanto não houver foto, o site mostra um marcador elegante com a marca — nada parte. 💛
